# RUNDINORUN

**RUNDINORUN** is a Chrome T-Rex Runner style game built in python using pygame modules.

# About
- The following represents a recreated version of the famous Dinosaur Game from Chrome Browser using Python and PyGame.
- A simple and easy-to-use GUI is provided for better gameplay.
- Using custom sound and sprites that original lacks.
- Can be played offline ;) 
## Installation

Currently build is only available for Windows/Linux systems, you can download it from releases page.

**Other Systems:**
clone the repository and run the below command (python and pygame module should be installed first).
```python
python main.py
```
